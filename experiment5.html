ใบงานการทดลองที่ 5

การเรียกดูข้อมูลขั้นสูง เรียกดูข้อมูลจากหลายตาราง

ความรู้เบื้องต้น

วัตถุประสงค์ในการทดลอง

  1. ผู้เรียนสามารถใช้คำสั่ง INNER JOIN ในการเรียกดูข้อมูลจากหลายตารางได้
  2. ผู้เรียนสามารถใช้คำสั่ง LEFT JOIN ในการเรียกดูข้อมูลจากหลายตารางได้
  3. ผู้เรียนสามารถใช้คำสั่ง RIGHT JOIN ในการเรียกดูข้อมูลจากหลายตารางได้
  4. ผู้เรียนสามารถใช้คำสั่ง FULL JOIN ในการเรียกดูข้อมูลจากหลายตารางได้

ทฤษฎีก่อนการทดลอง


    การเรียกดูข้อมูลขั้นสูง เรียกดูข้อมูลจากหลายตาราง คือ การเรียกดูข้อมูลจากหลายตารางในครั้งเดียว เป็นส่วนหนึ่งในการเรียกดูข้อมูลขั้นสูง ในปัจจุบันฐานข้อมูลที่นิยมใช้มากที่สุดจะเป็นฐานข้อมูลแบบ Relational Database ซึ่งข้อมูลจะถูกเก็บอยู่ในตารางเป็นหมวดหมู่ และเชื่อมโยงความสัมพันธ์เข้าด้วยกัน ดังนั้นถ้าต้องการเรียกดูข้อมูลหลายๆอย่างในคราวเดียวกัน ข้อมูลต่างๆนั้นอาจกระจายอยู่ในหลายๆตาราง เราจึงต้องใช้คำสั่งเรียกดูข้อมูลจากหลายตารางเพื่อให้ได้ข้อมูลที่ต้องการออกมา ซึ่งในภาษา SQL จะมีคำสั่งรองรับการเรียกดูข้อมูลจากหลายตารางออกมาในครั้งเดียวอยู่ 4 คำสั่งได้แก่ INNER JOIN, LEFT JOIN, RIGHT JOIN และ OUTER JOIN ซึ่งแต่ละคำสั่งจะมีจุดเด่นและข้อดีในการเรียกดูฐานข้อมูลที่แตกต่างกันไป
    Primary Key คือ คอลัมน์ที่ไม่สามารถมีข้อมูลซ้ำกันได้ และถูกตั้งให้เป็นคอลัมน์หลักในการใช้อ้างอิงถึงข้อมูลในแถวนั้นๆ โดยทั่วไปมักจะเป็นคอลัมน์ id หรือ running number
    Foreign Key  คือ คอลัมน์ที่เก็บข้อมูลของ Primary Key ของตารางอื่นไว้ แต่ข้อมูลที่เก็บไว้ในคอลัมน์นั้นสามารถซ้ำกันได้ ใช้เป็นตัวอ้างอิงถึง Primary Key ของตารางอื่นตอนทำการเรียกดูข้อมูลจากหลายตาราง
    การใช้จุด( . ) คือ การอ้างถึงคอลัมน์ที่อยู่ในตารางนั้นๆ มักใช้ตอนชื่อคอลัมน์ของทั้งสองตารางซ้ำกัน จึงต้องระบุด้วยว่าคอลัมน์นั้นมาจากตารางใด เช่น ตาราง Employee มีคอลัมน์ Dep_id และตาราง Department ก็มีคอลัมน์ Dep_id เช่นเดียวกัน ถ้าต้องการอ้างถึงคอลัมน์ Dep_id ที่อยู่ในตาราง Employee จะเขียนได้ดังนี้ Employee.Dep_id แต่ถ้าต้องการอ้างถึงคอลัมน์ Dep_id ที่อยู่ในตาราง Department จะเขียนได้ดังนี้ Department.Dep_id

ขั้นตอนการทดลอง

INNER JOIN

    เป็นการจับเอา 2 ตารางมารวมกัน โดยยึดหลักว่าข้อมูลใน Primary Key กับ Foreign Key จะต้องตรงกันทั้ง 2 ตารางถึงจะนำข้อมูลนั้นมาแสดง

Syntax:การเรียดูข้อมูลขั้นสูง โดยใช้ INNER JOIN
SELECT * FROM table1 INNER JOIN table2 ON table1.foreign_key = table2.primary_key;
หมายเหตุ

- table1 หมายถึง ชื่อตารางที่ 1

- table2 หมายถึง ชื่อตารางที่ 2

Step 1.ใช้คำสั่งเพื่เรียกดูข้อมูลทั้งหมดภายในตาราง sales;

select * from sales;
