ใบงานการทดลองที่ 3

เรื่อง DML (Data Manipulation Language)

ความรู้เบื้องต้น

วัตถุประสงค์ในการทดลอง

  1. เพื่อให้ผู้เรียนสามารถเพิ่มข้อมูลลงไปในตารางได้
  2. เพื่อให้ผู้เรียนสามารถแก้ไขข้อมูลภายในตารางได้
  3. เพื่อให้ผู้เรียนสามารถลบข้อมูลภายในตารางได้

ทฤษฎีก่อนการทดลอง


DML (Data Manipulation Language) เป็นรูปแบบคำสั่งของภาษา SQL ที่เอาไว้จัดการเกี่ยวกับข้อมูลที่อยู่ในตาราง ซึ่งจะใช้คำสั่ง insert update และ delete หมายถึง เพิ่มข้อมูลลงในตาราง, แก้ไขข้อมูลในตาราง และ ลบข้อมูลในตาราง ตามลำดับ โดยข้อมูลที่จะเพิ่มลงในตารางจะเรียกว่า value การเพิ่ม value ลงใน column นั้น จะต้องดูด้วยว่าคอลัมน์นั้นเก็บข้อมูลประเภทใด ถ้าเป็นประเภทข้อความ value ที่จะเพิ่มลงไปนั้น จะต้องใส่เครื่องหมายฟันหนู(“”) ครอบ value ด้วย แต่ถ้าเป็นประเภทตัวเลข value ที่จะเพิ่มลงไปนั้นไม่จำเป็นต้องใส่เครื่องหมายใดๆครอบ value

INSERT

คือ การเพิ่มข้อมูลลงในตารางที่เราต้องการ โดยจะมีวิธีการ INSERT ข้อมูลอยู่ 2 แบบดังนี้

1. การเพิ่มข้อมูลลงในตารางแบบเรียงลำดับ หมายถึง การเพิ่ม value ลงในตาราง ผู้ใช้ ต้องรู้ลำดับของคอลัมน์ในตาราง เพราะตอน INSERT ลำดับของ value จะตรงกับลำดับขอคอลัมน์ที่เราจะเพิ่มข้อมูลลงไป

INSERT INTO table-name VALUES (value1, value2, value3, …) ;
หมายเหตุ

table-name หมายถึง ชื่อตาราง

value หมายถึง ข้อมูล

ขั้นตอนการทดลอง

INSERT การเพิ่มข้อมูลลงในตาราง

Step 1.ใช้คำสั่งเพื่เรียกดูข้อมูลทั้งหมดภายในตาราง categories

select * from categories;

Step 2.เพิ่มข้อมูลลงในตาราง categories โดยการเพิ่มข้อมูล CategoryID ="16" และ CategoryName ="เฟอร์นิเจอร์ห้องน้ำ"

INSERT INTO categories VALUES("16","เฟอร์นิเจอร์ห้องน้ำ");

Step 3.เรียกดูตาราง categories อีกครั้ง เพื่อตรวจสอบการเปลี่ยนแปลง

1.จงเพิ่มข้อมูลลงในตาราง products โดยมีข้อมูลดังนี้
- รหัสสินค้า(PID) คือ P09
- ชื่อสินค้า(ProductName) คือ เคาน์เตอร์
- รหัสสินค้า(CategoryID) คือ 16
- ราคา(Price) คือ 200
- จำนวน(InStock) คือ 10
- จำนวนการสั่ง(ReOrderPoint) คือ 3
- ลดราคา(DiscountLimit) คือ 0.1
- บริษัท(SupID) คือ 04

UPDATE การแก้ไขข้อมูลในตาราง

เป็นคำสั่งที่เอาไว้ใช้สำหรับการแก้ไขข้อมูลในฐานข้อมูล

โดยใส่คำว่า Update เข้าไปแล้วตามด้วยชื่อตารางที่ต้องการ update ข้อมูล จากนั้นใส่คำว่า set เพื่อเลือกคอลัมน์ที่ต้องการแก้ไขข้อมูลและใส่ค่าข้อมูลใหม่ลงไป เราสามารถใส่หรือแก้ไขได้หลายคอลัมน์ในคำสั่งเดียว จากนั้นใส่ where เพื่อระบุว่าให้เซตที่กรณีไหน

ต้องการแก้ไขข้อมูลในตาราง catagories โดยเปลี่ยน ชื่อ categories เฟอร์นิเจอร์ห้องน้ำ เป็น เฟอร์นิเจอร์ทั่วไป

Step 1.ใช้คำสั่งเพื่เรียกดูข้อมูลทั้งหมดภายในตาราง categories

select * from categories;

Step 2.แก้ไขข้อมูลลงในตาราง categories โดยการแก้ไขข้อมูล CategoryName ="เฟอร์นิเจอร์ทั่วไป" โดยมี CategoryID="16"

UPDATE categories SET CategoryName ="เฟอร์นิเจอร์ทั่วไป" WHERE CategoryID="16";

Step 3.เรียกดูตาราง categories อีกครั้ง เพื่อตรวจสอบการเปลี่ยนแปลง

2.จงแก้ไขข้อมูลลงในตาราง products โดยเปลี่ยนแปลงข้อมูล รหัสสินค้า(PID) คือ P09 ดังนี้
- ชื่อสินค้า(ProductName) แก้ไขเป็น แจกันดอกไม้สไตล์ยุโรป
- ราคา(Price) แก้ไขเป็น 150
- บริษัท(SupID) แก้ไขเป็น 12

DELETE การลบข้อมูลทั้งหมดในตาราง

คำสั่งในการลบข้อมูลในตารางข้อมูล โดยสามารถใช้งานรวมกับคำสั่ง WHERE เพื่อสร้างเงื่อนไขสำหรับการลบข้อมูล

ต้องการลบข้อมูลในตาราง catagories โดยลบข้อมูลที่มี CategoryID="16"

Step 1.ใช้คำสั่งเพื่เรียกดูข้อมูลทั้งหมดภายในตาราง categories

select * from categories;

Step 2.ลบข้อมูลลงในตาราง categories โดยการลบข้อมูล CategoryID="16" ดังนี้

DELETE FROM categories WHERE CategoryID="16";

Step 3.เรียกดูตาราง categories อีกครั้ง เพื่อตรวจสอบการเปลี่ยนแปลง

3.จงลบข้อมูลลงในตาราง products ที่มี รหัสสินค้า(PID) คือ P09

1.จงเขียนคำสั่งเพิ่มข้อมูลลูกค้าลงในตาราง customers โดย มีข้อมูลดังนี้ นาย ไกรลาศ แสงสุข ที่อยู่ 28 ถ.เลียบคลองแม่ริม อ.แม่ริม อยู่จังหวัดเชียงใหม่ ภาคเหนือ รหัสไปรษณีย์ 53010 เบอร์โทรติดต่อ 0890070030 PaymentPeriod="0" รหัสลูกค้า C0011



2. จากข้อที่ 1 จงแก้ไขข้อมูล เป็น นาย เศรษฐี ตลอดไป ที่อยู่ 28 ถ.เลียบคลองแม่ริม อ.แม่ริม อยู่จังหวัดเชียงใหม่ ภาคเหนือ รหัสไปรษณีย์ 53010 เบอร์โทรติดต่อ 0898889999 PaymentPeriod="15"



3. จากข้อ 2 จงลบข้อมูลของ นาย เศรษฐี ตลอดไป
