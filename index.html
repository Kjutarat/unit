ข้อมูลเบื้องต้น

วัตถุประสงค์ในการทดลอง

  1. เพื่อให้ผู้เรียนสามารถเรียกดูฐานข้อมูลทั้งหมดได้
  2. เพื่อให้ผู้เรียนสามารถเลือกใช้ฐานข้อมูลได้
  3. เพื่อให้ผู้เรียนสามารถเรียกดูตารางทั้งหมดในฐานข้อมูลได้
  4. พื่อให้ผู้เรียนสามารถเรียกดูโครงสร้างของตารางได้

ความรู้เบื้องต้น

ภาษา SQL เป็นภาษาที่ใช้จัดการเกี่ยวกับฐานข้อมูล การเขียนคำสั่ง SQL หลังจากเขียนคำสั่งนั้นจบแล้วต้องตามด้วยเครื่องหมาย ;(เซมิโคลอน) เพื่อเป็นการจบประโยคคำสั่งนั้นๆ ภาษา SQL เป็นภาษาที่ไม่จัดอยู่ในกรณี case sensitive หมายถึง คำสั่ง SQL สามารถใช้ตัวอักษรพิมพ์ใหญ่หรือพิมพ์เล็กก็ได้ เช่น Select มีความหมายเดียวกับ select หรือ SeLeCT หรือ SELECT เป็นต้น

เริ่มการทดลอง

SHOW DATABASES

คำสั่งที่ใช้สำหรับเรียกดูฐานข้อมูลทั้หมดที่มีอยู่ โดยรูปแบบคำสั่งจะใช้คำสั่ง SHOW แล้วตามด้วยสิ่งที่เราต้องการจะแสดงเช่นฐานข้อมูล DATABASES

SHOW DATABASES;
