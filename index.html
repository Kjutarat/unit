ความรู้เบื้องต้น

วัตถุประสงค์ในการทดลอง

  1. เพื่อให้ผู้เรียนสามารถเรียกดูฐานข้อมูลทั้งหมดได้
  2. เพื่อให้ผู้เรียนสามารถเลือกใช้ฐานข้อมูลได้
  3. เพื่อให้ผู้เรียนสามารถเรียกดูตารางทั้งหมดในฐานข้อมูลได้
  4. พื่อให้ผู้เรียนสามารถเรียกดูโครงสร้างของตารางได้

ทฤษฎีก่อนการทดลอง


ระบบฐานข้อมูลเชิงสัมพันธ์ (Relational Database)เป็นฐานข้อมูลที่ใช้โมเดลเชิงสัมพันธ์ (Relational Database Model) ซึ่งผู้คิดค้นโมเดลเชิงสัมพันธ์นี้คือ Dr. E.F. Codd โดยใช้หลักพื้นฐานทางคณิตศาสตร์ เนื่องด้วยแนวคิดของแบบจำลองแบบนี้มีลักษณะที่คนใช้กันทั่วกล่าวคือมีการเก็บเป็นตาราง ทำให้ง่ายต่อการเข้าใจและการประยุกต์ใช้งาน ด้วยเหตุนี้ ระบบฐานข้อมูลแบบนี้จึงที่ได้รับความนิยมมากที่สุด ในแง่ของ entity แบบจำลองแบบนี้คือ แฟ้มข้อมูลในรูปตาราง และ attribute ก็เปรียบเหมือนเขตข้อมูล ส่วนความสัมพันธ์คือความสัมพันธ์ระหว่าง entity

ฐานข้อมูลเชิงสัมพันธ์ คือ การเก็บข้อมูลในรูปของตาราง (Table) หลายๆตารางที่มีความสัมพันธ์กัน ในแต่ละตารางแบ่งออกเป็นแถวๆ และในแต่ละแถวจะแบ่งเป็นคอลัมน์ (Column) ในทางทฤษฎีจะมีคำศัพท์เฉพาะแตกต่างออกไป เนื่องจากแบบจำลองแบบนี้เกิดจากทฤษฎีทางคณิตศาสตร์เรื่องเซ็ท (Set)

ภาษา SQL เป็นภาษาที่ใช้จัดการเกี่ยวกับฐานข้อมูล การเขียนคำสั่ง SQL หลังจากเขียนคำสั่งนั้นจบแล้วต้องตามด้วยเครื่องหมาย ;(เซมิโคลอน) เพื่อเป็นการจบประโยคคำสั่งนั้นๆ ภาษา SQL เป็นภาษาที่ไม่จัดอยู่ในกรณี case sensitive หมายถึง คำสั่ง SQL สามารถใช้ตัวอักษรพิมพ์ใหญ่หรือพิมพ์เล็กก็ได้ เช่น Select มีความหมายเดียวกับ select หรือ SeLeCT หรือ SELECT เป็นต้น

สรุปแล้วคำสั่ง SQL สามารถเขียนแบบบรรทัดเดียวหรือหลายบรรทัดก็ได้ ซึ่งผลลัพธ์ที่ได้จะเหมือนกัน แต่เพื่อให้อ่านง่ายจึงนิยมเขียนแยกคำสั่งออกเป็นคำสั่งละ 1 บรรทัด เครื่องหมาย " ; " หลังประโยคคำสั่ง SQL ใช้เพื่อสิ้นสุดประโยคคำสั่งและเพื่อแยกแต่ละคำสั่งออกจากกัน

ขั้นตอนการทดลอง

คำสั่งเรียกดูฐานข้อมูล

คำสั่งที่ใช้สำหรับเรียกดูฐานข้อมูลทั้งหมดที่มีอยู่ โดยรูปแบบคำสั่งจะใช้คำสั่ง SHOW แล้วตามด้วยสิ่งที่เราต้องการจะแสดงเช่นฐานข้อมูล DATABASES

SHOW DATABASES;

ให้นักเรียนทดลองใช้คำสั่งเรียกดูฐานข้อมูลว่ามีฐานข้อมูลชื่ออะไรบ้าง

คำสั่งเลือกใช้งานฐานข้อมูล

คำสั่งสำหรับเลือกใช้ฐานข้อมูลที่มีอยู่ โดยพิมพ์คำสั่ง Use ตามด้วยชื่อฐานข้อมูล

USE DatabasesName;

ให้นักเรียนทดลองใช้คำสั่งเรียกดูฐานข้อมูลว่ามีฐานข้อมูลชื่ออะไร

คำสั่งเรียกดูตารางในฐานข้อมูล

คำสั่งสำหรับเลือกใช้ฐานข้อมูลที่มีอยู่ โดยพิมพ์คำสั่ง Use ตามด้วยชื่อฐานข้อมูล

USE ชื่อฐานข้อมูล;

คำสั่งเรียกดูโครงสร้างของตาราง

USE ชื่อฐานข้อมูล;
