ทฤษฎีก่อนการทดลอง


    การใช้คำสั่ง SQL FUNCTION เพื่อเรียกดูข้อมูล เพื่อความสะดวกต่อการคิวรี่จึงการการเรียกใช้ฟังก์ชั่นเหล่านี้ให้พร้อม ใข้งานได้ทันที ผู้ใช้งานจึงควรทราบว่าจะใช้งานอย่างไรและผลลัพธ์ที่ได้จะเป็นอย่างไร โดยสามารถจัดกลุ่มของฟังก์ ชัน ออกเป็น 6 กลุ่มคือ

1) Operator
2) Control Flow Function
3) String Function
4) Numeric Function
5) Data and Time Function
6) Group Function

Operator

    คือกลุ่มของเครื่องหมายต่างๆ สามารถแบ่งออกได้อีก 3 ประเภท คือ

• Arithmetic Operator (เครื่องหมายทางคณิตศาสตร์)
• Comparison Operator (เครื่องหมายเปรียบเทียบ)
• Logic Operator (เครื่องหมายทางตรรกะ)

Control Flow Functions

    เป็นกลุ่มของฟังก์ชันสำหรับควบคุมการไหลของข้อมูล ได้แก่ กลุ่มคำสั่งตัดสินใจ หรือทางเลือก มี 2 ฟังก์ชันที่สำคัญคือ ฟังชันก์ IF และ CASE

รูปแบบฟังก์ชัน IF :
IF (e1, e2, e3)
หมายเหตุ

   e1 คือ เงื่อนไขที่ใช้ในการเปรียบเทียบ
   e2 คือ ถ้า e1 เป็นจริงจะทำคำสั่ง e2
   e3 คือ ถ้า e1 เป็นเท็จจะทำคำสั่ง e3

รูปแบบฟังก์ชัน CASE :
CASE
WHEN [condition] THEN result
[WHEN [condition] THEN result]
[ELSE result]
หมายเหตุ

   condition คือ เงื่อนไขที่ใช้ในการเปรียบเทียบ
   WHEN condition THEN result คือคำสั่งที่ต้องทำ โดยเปรียบเทียบว่าตรงกับ case ไหนก็ทำ case นั้น
   ELSE result คือ คำสั่งที่ต้องทำโดยเปรียบเทียบแล้วไม่ตรงกับ case ไหนเลย

String Functions

    คือ กลุ่มของฟังก์ชันที่ใช้ในการจัดการข้อมูลที่เป็นตัวอักขระ หรือตัวอักษร

Function
ความหมาย
concat(s1,s2…)
ใช้สำหรับการต่อสตริง s1,s2… เข้าเป็นสตริงเดียว
length(s)
ใช้หาค่าความยาวของสตริง s
lower(s)
แปลงสตริง s เป็นตัวอักษรเล็ก
upper(s)
แปลงสตริง s เป็นตัวอักษรใหญ่
replace(target,find,replace)
ใช้หาข้อมูลและแทนที่ โดย target คือ ฟิลด์ที่จะทำการค้นหา find คือข้อมูลที่ค้นหา และ replace คือข้อมูลที่จะใช้แทนที่
substring(s,position,length)
ใช้สำหรับแยกสตริง s โดย position คือตำแหน่งเริ่มต้นที่แยกสตริง และ length คือ ความยาวที่ต้องการใช้สำหรับตัด white space ออกจากสตริงทั้งด้านหน้าและด้านหลัง
trim(s)
ใช้สำหรับตัด white space ออกจากสตริงทั้งด้านหน้าและด้านหลัง
Itrim(s)
ใช้สำหรับตัด white space ออกจากสตริงทางด้านหน้า
rtrim(s)
ใช้สำหรับตัด white space ออกจากสตริงทางด้านหลัง
like
ใช้สำหรับเปรียบเทียบตัวอักษร
strcmp(s1,s2)
ใช้สำหรับเปรียบเทียบสตริง 2 ชุด
โดยถ้าค่าเป็น 0 แสดงว่า สตริง s1 และ s2
เท่ากัน ถ้าค่าเป็น 1 แสดงว่า สตริง s1 มากกว่า s2
ถ้าค่าเป็น -1 แสดงว่า สตริง s1 น้อยกว่า s2

Numeric Functions

    คือ กลุ่มของฟังก์ชันที่ใช้ในการจัดการข้อมูลที่เป็นตัวอักขระ หรือตัวอักษร

Function
ความหมาย
abs(n)
ใช้สำหรับหาค่าสัมบูรณ์ของตัวเลข
ceiling(n)
ใช้สำหรับปัดจุดทศนิยมทิ้งและเพิ่มค่าจำนวนเต็มขึ้น 1
floor(n)
ใช้สำหรับปัดจุดทศนิยมทิ้ง ค่าจำนวนเต็มเท่าเดิม
n div m
ใช้สำหรับหา n หาร m ผลลัพธ์เป็นผลหาร ไม่เอาเศษ
mod(n,m)
ใช้สำหรับหา n หาร m ผลลัพธ์เป็นเศษจากการหาร
power(n,m)
ใช้สำหรับหาเลขยกกำลัง n ยกกำลัง m
rand(n)
ใช้สำหรับหาเลขสุ่ม 0-1 ค่า n อาจจะไม่ต้องระบุก็ได้ ถ้าระบุจะเป็น seed value ( หมายถึงการสุ่มด้วยตัวเลข n เดียวกันจะมีค่าเท่ากัน )
round(n[,d])
ใช้สำหรับปัดตัวเลขทศนิยม n ตัวเลข d คือจำนวนทศนิยมที่ต้องการซึ่งไม่ระบุก็ได้จะเป็นการปัดเป็นเลขจำนวนเต็ม ถ้าตัวเลขถัดจากตำแหน่งที่ต้องการปัดมีค่าน้อยกว่า 5 จะไม่ปัด แต่ถ้าตัวเลขถัดจากตำแหน่งที่ต้องการปัดมีค่ามากกว่าหรือเท่ากับ 5 จะปัดขึ้น
sqrt(n)
ใช้สำหรับหาค่ารากที่ 2 ของ n

Date and Time Functions

    คือ กลุ่มของฟังก์ชันที่ใช้ในการจัดการข้อมูลที่เป็นเกี่ยวกับวันที่และเวลา

Function
ความหมาย
curdate( ),curtime( ) now()
ใช้สำหรับแสดงวันเดือนปี ปัจจุบันและเวลาปัจจุบัน
date_format(date,format)
 ใช้สำหรับแสดงวันเดือนปี และเวลาปัจจุบัน ใช้สำหรับแสดงวันเดือนปี ตาม format ที่กำหนด
   %W คือ วันในสัปดาห์เป็นภาษาอังกฤษ Sunday-Saturday
   %w คือ วันในสัปดาห์เป็นตัวเลข 0-6
   %D คือ วันที่และลำดับ
   %d คือ วันที่
   %M คือ เดือนเป็นภาษาอังกฤษ January-December
   %m คือ เดือนเป็นตัวเลข 0-12
   %Y คือ ปีเป็นตัวเลข 4 หลัก
   %y คือ ปีเป็นตัวเลข 2 ตัวท้าย
time_format(time,format)
 ใช้สำหรับแสดงเวลาตาม format ที่กำหนด
  %H คือ จำนวนชั่วโมง 00-23
  %h คือ จำนวนชั่วโมง 01-12
   %i คือ นาที 01-59
  %S คือ วินาที 01-59
datediff(date1,date2)
 ใช้สำหรับหาว่าวันเดือนปี 2 ชุด ห่างกันกี่วัน

Group Functions

    เป็นกลุ่มของฟังก์ชันที่ใช้จัดแบ่งข้อมูลออกเป็นกลุ่มตามเงื่อนไขที่กำหนดในคำสั่ง

Function
ความหมาย
avg(field)
 ใช้สำหรับหาค่าเฉลี่ย
count(field)
 ใช้สำหรับนับจำนวนของข้อมูล
min(field)
 ใช้สำหรับหาค่าที่น้อยที่สุด
max(field)
 ใช้สำหรับหาค่าที่มากที่สุด
std(field)
 ใช้สำหรับหาค่าเบี่ยงเบนมาตรฐาน
sum(field)
 ใช้สำหรับหาผลรวม
