ทฤษฎีก่อนการทดลอง


DDL (Data Definition Language)เป็นภาษาสำหรับนิยามโครงสร้างฐานข้อมูลเพื่อสร้าง เปลี่ยนแปลง หรือลบโครงสร้างฐานข้อมูลเรียกได้ว่าเป็น “Schema” ใช้บอกว่าในฐานข้อมูลประกอบ ด้วยตารางใด แต่ละตารางมีความสัมพันธ์กันอย่างไร และในตารางมี Primary Key อะไร โดยมีฟังก์ชั่น การสร้าง แก้ไข ลบตาราง การวิวข้อมูล การสร้างระบควบคุมความปลอดภัยของฐานข้อมูล การสร้างดัชนีเพื่อการเข้าถึงตารางอย่างรวดเร็ว และควบคุมการจัดเก็บข้อมูลด้วย DBMS เช่น คำสั่ง Create Alter Drop จะใช้สำหรับการ สร้างปรับเปลี่ยนและลบโครงสร้างฐานข้อมูลหรือโครงสร้าง ของตาราง โดยคำสั่งประเภทนี้จะทำกับโครงสร้างของฐานข้อมูลเท่านั้น ไม่มีผลกับข้อมูลใน Field ใดๆ ซึ่งคำสั่งในการแก้ไขข้อมูลจะอยู่ในประเภทของ DML(Data Manipulation Language)

ประเภทของข้อมูลในฐานข้อมูล MySQL คือ ในการสร้างตารางภายในฐานข้อมูล เราจำเป็นจะต้องกำหนดประเภทฟิลด์ของฐานข้อมูล โดยสิ่งที่ควรทราบคือ ประเภทของข้อมูลต่างๆ ของ MySQL มีประเภทข้อมูลอะไรบ้าง เพื่อที่เราจะได้กำหนดประเภทข้อมูล ได้สอดคล้องกับข้อมูลที่เราจะจัดเก็บ

การสร้างฐานข้อมูล

ใช้สร้างฐานข้อมูลที่ต้องการ โดยใส่ชื่อฐานข้อมูลหลัง Syntax

CREATE DATABASE DATABASE-NAME ;
DATABASE-NAME คือ ชื่อของฐานข้อมูล

ทดลองการสร้างฐานข้อมูลชื่อ ex_540303xx_test

Create Database ex_540303xx_test;

1.จงสร้างฐานข้อมูลชื่อ ex_รหัสนักศึกษา_test

การลบฐานข้อมูล

ใช้ลบฐานข้อมูล โดยการใส่ชื่อฐานข้อมูลที่ต้องการลบ โดยพิมพ์คำสั่ง DROP DATABASE ชื่อของฐานข้อมูล;

DROP DATABASE DATABASE-NAME ;
DATABASE-NAME คือ ชื่อของฐานข้อมูล

Drop Database ex_540303xx_test;

2.ทดลองลบฐานข้อมูล ชื่อ "ex_รหัสนักศึกษาของตนเอง_test;"

ใช้คำสั่ง Show Database ex_รหัสนักศึกษาของตนเอง; เพื่อตรวจสอบ พร้อมบันทึกผลการทดลอง

การสร้างตาราง (Table)

ใช้สร้างตารางลงในฐานข้อมูล พร้อมกำหนดคอลัมน์ และชนิดของข้อมูล ในแต่ละฟิลด์นั้น

CREATE TABLE table_name
( column_name_1 column_type,
column_name_2 column_type,
.....,
column_name_n column_type ,
[PRIMARY KEY < column_name >],
[FOREIGN KEY < column_name > REFERENCES ]);
หมายเหตุ

table_name คือ ชื่อของตารางที่ต้องการสร้าง
column name คือชื่อของคอลัมน์
column type คือชนิดของข้อมูล
Link_table คือตารางปลายทางที่ต้องการเชื่อมโยง Foreign key ไป
CREATE TABLE person
(studentId int(8) not null,
first_name varchar(50),
last_name varchar(50),
TelNo int(20));

3.ทดลองคำสั่งสร้างตาราง ใช้คำสั่ง Show tables; บันทึกผลการทดลอง

การกำหนด Primary Key

การกำหนด primary key สามารถกำหนดได้สองรูปแบบ

     กำหนด Primary Key ไว้หลังคอลัมน์ที่ต้องการกำหนดให้เป็น Primary key

สร้าง ตาราง Address มีข้อมูลดังนี้

CREATE TABLE Address
(studentId int(8) primary key,
Address varchar(200),
TelNos int(20));

ใช้คำสั่ง describe Address;

      กรณีที่ตารางใช้คอลัมน์ หลายตัวร่วมกันเพื่อทำหน้าที่ Primary Key ต้องเขียนในรูปแบบกำหนด Primary key ไว้หลังจากกำหนดคอลัมน์อื่นแล้ว ตัวอย่าง

CREATE TABLE Address
(studentId int(8),
Address varchar(200),
TelNo int(20),

primary key (studentId));

ทดลองใช้คำสั่ง Drop table สร้าง ตาราง Address อีกครั้ง

4.บันทึกผลการทดลอง

การกำหนด Foreign Key

    คือ คอลัมน์ที่กำหนดเพื่อใช้อ้างอิงไปยัง Primary Key ในตารางอื่น โดยทำให้ตารางสองตารางที่ ต่างกัน มีความสัมพันธ์กัน

Foreign Key คอมลัมน์ที่ต้องการลิ้งค์ Refernces ตารางที่ต้องการเชื่อม(คอมลัมน์ที่ต้องการลิ้งค์);
หมายเหตุ

ต้องสร้างตารางที่ต้องการเชื่อมโยงก่อนจึงจะสร้าง Foreign Key ได้

    ใช้คำสั่ง Drop table person;

การกำหนด Foreign Key แบบท้ายคอลัมน์ที่ต้องการกำหนด
CREATE TABLE person
( studentId int(8) primary key,
first_name varchar(50),
last_name varchar(50) ,
TelNo int (20)References Address(TelNo));

5.    ใช้คำสั่ง Describe person; บันทึกผลการทดลอง

DROP TABLE table_name;
หมายเหตุ

table_name  คือ ชื่อตาราง

การเปลี่ยนชื่อตาราง (Table)

การเพิ่มคอลัมน์ ในตาราง (Table)

    การเพิ่มคอลัมน์ในตารางเราจะใช้คำสั่ง alter ตามดัง syntax ข้างล่างระบุชื่อคอลัมน์ และจำเป็นต้องระบุชนิดของข้อมูล

ALTER TABLE table_name ADD column_name column_type;
หมายเหตุ

    ดูประเภทชนิดของข้อมูลที่ต้องการกำหนดได้จากตารางรายละเอียดประเภทข้อมูลของ MySQL
    - table_name คือ ชื่อตาราง
    - Column name คือ ชื่อคอลัมน์
    - column type คือ ชนิดของข้อมูล

ใช้คำสั่ง Describe person;

ทดลองคำสั่งเพิ่มตาราง person และ เพิ่มคอลัมน์ชื่อ sex ชนิดข้อมูลคือ smallint(1)

ALTER TABLE person ADD sex smallint(1);

การลบคอลัมน์ ในตาราง (Table)

    การลบคอลัมน์ในตารางเป็นการลบข้อมูลใน field ทั้งหมด ที่ระบุชื่อคอมลัมน์ที่ต้องการลบและลบคอลัมน์นั้นออกไปอย่างสิ้นเชิง วิธีการลบคอมลัมน์ โดยใช้ ทำสั่ง alter นั้น จะต้องระบุชื่อฐานข้อมูลของคอลัมน์ที่ต้องการลบออก และระบุชื่อคอลัมน์ ที่ต้อง การลบตาม syntax ด้านล่าง

ALTER TABLE table_name DROP column_name;
หมายเหตุ

table_name คือ ชื่อตาราง

column_name คือ ชื่อคอลัมน์ที่ต้องการลบ

ALTER TABLE Person DROP MajorNo;

การสร้างและลบดัชนี (index)

   การสร้างดัชนี (index)

       การสร้าง index จะช่วยให้การค้นหาและการเข้าถึงข้อมูลได้รวดเร็วขึ้น โดยมีการจัดการเกี่ยวกับ การจัดเก็บข้อมูลลงฐานข้อมูล เช่นเดียวกับหนังสือที่มีหลายหน้าจึงต้องมีดัชนีในการค้นหาคำสำคัญ

CREATE INDEX index name ON table name
( column name1,column name2,...,column name n);
หมายเหตุ

index name คือ ชื่อ index
